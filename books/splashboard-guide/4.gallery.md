以下では、各 preset を日本語で一言ずつ補足します。

## Home preset ( 5 種 )

新しいシェルを開いたときに出る splash です ( CWD が git レポ root でなければ ) 。 `cd` で出るのは `cd $HOME` で $HOME 自体に降りたときだけで、それ以外の非 git ディレクトリへの `cd` では何も再描画されません。

- **`home_splash`**: ターミナル名の巨大な figlet ヒーローに postfx ( ノイズ / スキャンライン的な演出 ) が乗る構成。周辺に天気、月相、年の進捗が並ぶ
- **`home_daily`**: 時間帯ヒーロー ( "morning" / "afternoon" など ) を中央に、左に天気 / 日の出 / 月相、右に 年 / 月 / 週 / 日 の進捗ゲージ。朝に眺める想定
- **`home_github`**: アバター + ハンドル + contribution heatmap を中心に、 open PR / レビュー待ち / 最近のアクティビティの timeline 。 GitHub をよく使う人向け
- **`home_minimal`**: 静かな 3 行 preset 。時間帯の挨拶、日付 1 行、今日の一言。 figlet も chart もなし、余白多め
- **`home_feed`**: 朝の briefing 風。 Hacker News のトップ、 on-this-day 、空 ( 日の出 / 月相 ) 、 Wikipedia の今日の特集

最初に選ぶ目安として、毎朝シェルを開く習慣がある人は `home_daily` か `home_feed` 、 GitHub 中心なら `home_github` 、雰囲気重視なら `home_splash` が刺さりやすい組み合わせです。

## Project preset ( 4 種 )

git レポジトリのトップに `cd` したときに出る splash です ( レポジトリが `./.splashboard/` を同梱していない場合 ) 。

- **`project_splash`**: レポ名の巨大 figlet + postfx 、その下に slug / description / license 。レポ同梱の `dashboard.toml` 向けの構成
- **`project_github`**: 動的なレポヒーロー + サブタイトル、 branch 状態、 commit / CI の sparkline 、 open PR 、最近のリリース。開発中の状況を映す daily driver
- **`project_minimal`**: レポ名、最新タグ、最近 3 commit だけ。完全オフライン ( GitHub 呼び出しゼロ ) なので token 不要、画面も奪わない
- **`project_codebase`**: レポ解析ダッシュボード。中央のレポ名 figlet を、ディレクトリ別ファイル数と最大ファイルランキングが挟み、上に LOC / age / コメント密度、下に churn と TODO ホットスポット。これもローカル完結で network 不要

## テーマで空気が変わる

`splashboard install` の 3 番目の選択で選ぶテーマは、 26 種類 ( `default` 含む ) 同梱されています。代表的なものを挙げると、

- **`tokyo_night`**: 青〜紫のダーク系。落ち着いた色合いで日常使い向き
- **`catppuccin_mocha`**: パステル寄りのダーク。視認性と柔らかさのバランス
- **`dracula`**: 紫 / ピンク強めのコントラスト系。 sparkline や chart が映える
- **`nord`**: 冷たい青系。情報密度の高い preset と相性が良い
- **`gruvbox_dark`**: オレンジ / 黄寄りの暖色ダーク
- **`monokai`** / **`synthwave_84`** / **`rose_pine_dawn`** / **`night_owl`** などのバリエーション
