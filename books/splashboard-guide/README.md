## 文体ルール

Zenn 本の慣行と作者の好みに合わせる。

- **em-dash (—) は使わない**。 `。` で文を切るか、 `、` `:` `( )` で繋ぐ
- **「」 は最小限**。引用 / 概念名の強調は外し、本当の引用と固有概念名 (例: 状態名そのもの) だけ残す
- **装飾的な比喩は使わない**。 「目を引く」「看板」「派手枠」「本領」「意外と効きます」「楽しくなります」などは直接的な技術記述に置き換える
- **テンプレ的な煽り** ( 「ぜひ」 / 「楽しみにしています」 / 「お付き合いください」 ) は使わない
- 段落は短め (2-3 文) 、コードブロックで頻繁に区切る
- 強調 `**...**` は 1 段落 1-2 個まで、乱発しない
- 章末は「次章では」「次の章で」だけで終わらせる。煽らない

## 画像 / リンク規約

- **画像プレースホルダ** は `:::message` ブロックに 📸 / 🖼 / 🎞 絵文字 + `**TODO**:` + 何を入れるかの説明を書く。本物の画像が用意できたら `![alt](path)` に置換する。 現状 45 箇所
- **bare URL カード** ( Zenn の OG カード ) は重要な外部リンクを段落から独立した行に置く。各章 1-2 個まで、乱発しない。 現状 22 箇所
- **inline link** ( `[label](url)` ) は文中の補足リンクに使う。 bare URL とは役割を分ける
